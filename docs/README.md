# Auto-Sign Frontend (GitHub Pages)

This folder is published by GitHub Pages and provides the frontend UI.

## Setup

1) Deploy the Flask backend (app.py) to a public host (Render/Heroku/Railway/etc.).
2) Update `BACKEND_URL` in `docs/index.html` to your backend URL.
3) Commit and push to GitHub. Pages will redeploy automatically.

## Example backend URL

- https://your-app.onrender.com
